# 🤱 MaaCare AI - Postpartum Health Chatbot

An empathetic AI-powered postpartum health companion designed for new mothers in Bangladesh. Built entirely on **Botpress** (no-code).

## 🌟 Features

- 🗣️ Responds in **Bengali (বাংলা)**, **Banglish**, and **English**
- 🍲 Culturally relevant nutrition advice (shol mach, daal, laal shaak, etc.)
- 🧠 Empathetic persona — behaves like a trusted আপু (elder sister)
- 🚨 Emergency detection — surfaces helpline numbers (999, 16511) for crisis situations
- 📚 RAG-powered Knowledge Base with maternal health data
- ⚕️ Medical disclaimers on every health response

## 🛠️ Tech Stack

| Component | Tool |
|-----------|------|
| Chatbot Platform | [Botpress](https://botpress.com) |
| AI Engine | Botpress Autonomous Node (LLM-powered) |
| Knowledge Base | Botpress RAG (Retrieval-Augmented Generation) |
| Deployment | Botpress Cloud (free tier) |
